# How to run the application?

1. Use e.g. VS Community 2017
2. Required .NET FW 4.6.1
3. Build it (it should restore Nuget packages)
4. Please, use Chrome browser or other modern ones.
5. And just run BundleWeb project

# How to run the tests of the application
1. xUnit is being used
2. Go to Test->Run->All Tests (it should open Test Explorer and run UnitTests project)
